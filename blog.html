What's the difference between local storage and session storage?
Session storage :
  • store data in browser for a session and data will delete when tab is closed.
  • data not available for future sessions.
  • capacity is 5kb.
Local storage :
  • store data in browser without any expiration data and data will not delete when tab is closed.
  • data is available for future sessions.
  • capacity is 10kb.
What's is global scope and block scope in javascript?
Block scope :

A block is a piece of code inside a set of curly braces.if a variable declared with let/const inside a block scope this varibale will be available only inside its own block.if,else,for loop are example of block scope.

Global scope :

Out side of block scope and function scope is called global scope.if we declared a variable in global scope this variable will be available for any others scope.

How can we get undifined in javascript?

When we write javascript code, sometimes we get undefined error. Below I have mentioned some reasons why we get this error:

  • When a variable is declared but not initialized then the value of this variable will be undefined.
  • when directly set the value of a variable undefined.
  • when we console.log any function that didn't return anything.
  • when we don't pass argument for any parameter then that parameter value will be undefined.
  • when we write return inside a function but don't mention what to return then it will return undefined.
  • when we access any object property that doesn't exist will return undefined.
